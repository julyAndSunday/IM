# IM

基于netty的即时通讯系统，提供用户添加好友、私聊、群聊、文件转发、查看朋友圈等功能。


| 模块      | 功能                                      |
| --------- | ----------------------------------------- |
| IM-Api    | 基础api，注册登录，好友群组，朋友圈等功能 |
| IM-Chat   | 服务端，提供消息转发集群                  |
| IM-Client | 客户端，提供用户各种操作                  |
| IM-common | 工具包                                    |
